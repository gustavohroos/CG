# CG
## Trabalho 1

Para rodar o trabalho 1, execute o seguinte comando:
```
npm install
```
para instalar as dependências e
```
npx vite
```
para rodar o projeto.

Versões utilizadas:
- Node 20.5.0
- Vite 9.8.0
